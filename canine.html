Canine Heart Scoring Project

The goal of this project was to assist veterinarians in measuring the heart size of canines. This is an important job because an enlarged heart can indicate the presence of heart disease. The manual annotation process used by veterinarians is quite tedious, so my group applied machine learning techniques to assist in the process.

In this project, my role was to research a way to use machine learning to annotate the radiographs. To do this, I trained a version of the YOLO-Pose model from Ultralytics to predict the key annotation points. This involved creating a dataset of pre-annotated points, Formatting the data so that the model could train from it, and performing statistical analyses on the results. In the end, my annotation model achieved an accuracy of 75 percent over all the key points.
