Glass World Shipping Report Utility Web App

The goal of this project was to provide a convenient way to build a shipping report while workers are loading the delivery truck. This app allows anyone to connect via a web browser, and scan products using the camera built into their phone or tablet. The report generated from this app is then used to monitor the products leaving the shop and to make sure they arrive at the correct work site.

Some key features of this web app include highlighting products scanned that do not match the entered job name. This makes it convenient to see if a product is loaded on a truck going to a different site than intended. The app also saves the working report to session storage in the browser so a worker can refresh the page or load another webesite and still return to the page to continue the report where they left off. The app also makes a beep sound when a product code is scanned as an audible que.
