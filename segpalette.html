SegPalette Project

SegPalette is designed to provide the user with a simple palette-based recoloring tool while allowing the selection of specific segments to be recolored. To do this, SegPalette integrates the Segment Anything Model (SAM) from Meta AI to segment an uploaded image. Then the user can select a specific segment of the image to be recolored. Once the segment is selected, a color palette is generated. Then the user can change individual colors in the palette.

In this project, my role was to research color palette generation and recoloring methods. I then mocked up the palette generation in Matlab for a proof of concept. Once the bugs were worked out of the Matlab scripts, I implemented the color palette generation for the web app in JavaScript. This involved implementing a k-means clustering algorithm to generate the color palette.
