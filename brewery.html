Brewery Microservices Project

This is a personal project that I worked on to learn more about microservices. I wanted to gain experience developing microservices with Spring Boot framework in Java so I signed up for an online microservice course through Udemy. In this project, I implemented several RESTful APIs and performed unit testing on them. This project also gave me hands-on experience working with MySQL databases, and organizing dependencies using Maven. I even got to work a bit with Spring Cloud and Docker frameworks. Overall, this project gave me experience working on specific frameworks that I did not have the chance to use in school. It also gave me extra practice with code testing.
