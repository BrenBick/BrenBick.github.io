Projects

I have worked on many projects over the years but here are a few of my favorites.

Simply Green Android App

Developed an Android app to assist with the proper disposal of waste on campus at SFU. This project followed a typical Agile developement process involving 3 sprints, a project specifications document, user requirements, and unit testing.

SegPalette

Image segmentation, palette generator and recolouring web app. It incorporates the Segment Anything Model from Meta to segment an image. Then the user can select a segment to edit and a colour palette is generated. Finally, the user can change individual colors in the palette to edit the image.

  • Details

Shipping Report

Utility web app developed for Glass World. This app allows workers to scan cabinet products using their phones to generate a shipping report.

  • Details

Canine Heart Scoring

AI model trained to assist vereranarians with measuring canine heart size. The YOLO-Pose model from Ultralytics was applied to annotate the radiographs with an accuracy of 75%. Additionally, a CNN classifier was applied to the radiographs and acheived a 99% accuracy of predicting oversized hearts.

  • Details

Emotion Prediction from Video

Applied multimodal processing to predict emotion from video clips. Combined an LSTM for audio and a CNN for video in late fusion to learn the perceived emotions in video clips. Then the system output an emoji to visually communicate the predicted emotion.

  • Details

Brewery Microservices

Developed microservices using the Spring Boot Framework in Java to manage a brewery inventory. Used REST APIs to communicate with the database and used an object oriented design for the system. Unit tested the services using JUnit, AssertJ, and Mockito.
